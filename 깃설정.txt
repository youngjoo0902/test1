[깃 설치 및 설정 add, commit, push]
1. github.com 가입 > repository 추가(테스트용)
2. git-scm.com 에서 window용 git 클라이언트 다운로드 및 기본값 설치
3. cmd 에서 git --version 치면 깃 버전 확인할수 있다.
4. git config --global 쳐서 설정을 하는데, git config --global user.name 깃아이디 치면 해당 아이디로 전역 설정된다.
5. git config --global user.email 깃이메일 쳐서 설정한다.
6. 현재까지 깃을 사용할 수 있는 상태가 된다.
7. 어떤 폴더에 깃 연동을 할지, cmd 에서 경로 치고 들어가서 git clone https://깃주소 엔터치면 로컬repository 가 생성된다.
8. 또 cd 깃폴더 찾아들어가서, git add 파일명.확장명 치면 add가 된다.
9. git commit -m "커밋메세지" 입력하면 커밋이 된다. 아직까지 푸쉬를 안했기 때문에, 깃허브에 업로드가 안되어 있다.
10. git push 명령을 수행하면, 저장소로 올라간다.

[오픈소스 참여하는 이유]
1. 프로필에 한줄 더 넣으려고
2. 사회공헌
3. 어쩔수없이
4. 기타등등...

[깃 사용시 장점]
1. 동시에 여러사람이 협업하기 편하다.
2. 동시에 여러사람이 병렬적으로 개발하므로 개발속도가 빠르다.
3. 포폴 관리가 용이하다.
